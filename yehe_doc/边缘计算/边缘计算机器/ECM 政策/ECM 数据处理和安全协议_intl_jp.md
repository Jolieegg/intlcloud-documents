# 1\. **背景**
このモジュールは、エッジコンピューティングマシン（ECM）（「機能」）を使用する場合に適用されます。本モジュールは、[DPSA](https://intl.cloud.tencent.com/document/product/301/17347)のデータプライバシーとセキュリティ契約に組み込まれています。本モジュールで使用されているが定義されていない用語は、DPSAで定義された意味を持つものとします。DPSA と本モジュールの間に矛盾が生じた場合、その矛盾の範囲内で本モジュールが適用されるものとします。

# 2\. **処理**
当社は本機能に関連して、次のデータを処理します。

<table>
<tr><th style="width: 14%;">個人情報</th><th style="width: 42%;">用途</th></tr>
  <tr>
	<td><b>エラーログ</b>: UIN、APP ID、エラーメッセージ、現地時刻、ローカルIPアドレス </td>
    <td rowspan=2>このデータは、本機能の動作をモニタリングする目的で処理されます。</td>
   <tr>   
	<td><b>トランザクションログ</b>：UIN、APP ID、インターフェース名、結果、戻りコード、要求ID</td>
	</tr>
</table>





# 3\. **サービス地域**
DPSA で規定されている通り。

# 4\. **サブプロセッサー**
DPSA で規定されている通り。Aceville Pte. LtdおよびTencent Cloud Computing（北京）Co., Ltd. が含まれます。

# 5\. **データ保持**
当社は、本機能に関連して処理された個人データを以下のように保管します。

| **個人情報**                                     | **保持ポリシー**                                                      |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **エラーログ**|最大7日間保存されます |
|**トランザクションログ**|最大3か月保存されます。 |

お客様は DPSA に従って、当該個人データの削除を要請することができます。
