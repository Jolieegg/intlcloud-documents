This document describes how to modify the name of a resource pack.
## Prerequisites
You have purchased a resource pack. For more information, see [Purchasing Resource Pack](https://www.tencentcloud.com/document/product/1098/55247).

## Directions
1. Log in to the [TDSQL-C for MySQL console](https://console.cloud.tencent.com/cynosdb/mysql#/).
2. On the left sidebar, click **Resource Pack** to enter the resource pack management page.
3. Find the target resource pack directly on the page or quickly filter it out in the search box on the right, and click the edit icon under the **Resource Pack ID/Name**.
4. In the pop-up window for modifying the resource pack name, enter the new name of the resource pack and click **OK**.
>?The resource pack name can contain up to 60 letters, digits, or symbols (-_.).
