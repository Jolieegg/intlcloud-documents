## 작업 시나리오
인스턴스 실행 템플릿은 CVM 인스턴스 생성에 필요한 설정 정보(인스턴스 비밀번호 제외)를 저장합니다. 지정된 인스턴스 실행 템플릿을 사용하여 인스턴스를 빠르게 생성하고, 효율성과 사용자 경험을 향상시킬 수 있습니다. 본문은 CVM 콘솔을 통해 인스턴스 실행 템플릿을 생성, 관리 및 사용하여 인스턴스를 빠르게 생성하는 방법을 설명합니다.

## 사용 설명
- 인스턴스 실행 템플릿 생성 완료 후에는 설정 수정이 지원되지 않습니다.
- 인스턴스 실행 템플릿에 대해 하나 이상의 버전을 생성할 수 있으며, 각 버전에는 다른 구성 정보를 설정할 수 있습니다. 기본 버전을 지정하고 인스턴스 실행 템플릿을 사용하여 인스턴스를 생성할 때 기본 버전 설정을 사용할 수 있습니다.


## 작업 단계

### 인스턴스 템플릿 생성 및 조회
1. [CVM 콘솔](https://console.cloud.tencent.com/cvm/instance/index?rid=1)에 로그인하고 왼쪽 사이드바에서 **인스턴스 실행 템플릿**을 선택합니다.
2. ‘인스턴스 시작 템플릿’ 페이지에서 **템플릿 생성**을 클릭합니다.
3. ‘인스턴스 실행 템플릿’ 생성 페이지로 이동하여 ‘템플릿 이름’과 ‘템플릿 설명’을 입력합니다. 나머지 설정은 [구매 페이지를 통한 인스턴스 생성](https://intl.cloud.tencent.com/document/product/213/4855)을 참고하십시오.
4. 설정 정보 확인 단계에서 확장 클라우드=서비스=약관<>, 동의함을 선택하고 지금 만들기를 클릭합니다.
생성이 완료되면 콘솔에서 인스턴스 시작 템플릿을 찾을 수 있습니다. 
템플릿 ID를 클릭하여 템플릿 세부 정보 페이지에 들어가 구체적인 정보를 조회할 수 있습니다.


### 인스턴스 템플릿 버전 생성
1. ‘인스턴스 실행 템플릿’ 페이지에서 버전 템플릿 행의 오른쪽에 있는 **버전 생성**을 선택합니다. 
2. ‘인스턴스 실행 템플릿’ 생성 페이지로 이동한 후, [구매 페이지를 통한 인스턴스 생성](https://intl.cloud.tencent.com/document/product/213/4855)을 참고하여 설정합니다.
3. ‘설정 정보 확인’ 단계에서 ‘<Tencent Cloud 서비스 약관>을 읽고 이에 동의함’을 선택합니다.
**기존 설정 비교**를 선택하면 ‘기존 설정 비교’ 팝업 창에서 새 버전과 기존 인스턴스 실행 템플릿의 차이점을 확인할 수 있습니다. 
4. 오류가 없는지 확인한 후 **즉시 생성**을 클릭합니다.
생성 완료 후 ‘인스턴스 실행 템플릿’ 페이지에서 템플릿 행 앞의 <img src="https://main.qcloudimg.com/raw/bc4f9debb43518cdf0e1552f76f9b7fa.png" style="margin:-3px 0px">을(를) 클릭하여 펼쳐진 리스트에서 버전을 볼 수 있습니다.


### 인스턴스 템플릿의 기본 버전 지정
1. ‘인스턴스 실행 템플릿’ 페이지에서 템플릿 행 앞의 <img src="https://main.qcloudimg.com/raw/bc4f9debb43518cdf0e1552f76f9b7fa.png" style="margin:-3px 0px">을(를) 클릭합니다.
2. 펼쳐된 리스트에서 설정할 버전 오른쪽의 **기본값으로 설정**을 클릭합니다. 
3. ‘기본 템플릿 설정’ 팝업 창에서 **확인**을 클릭합니다.


### 인스턴스 템플릿을 사용하여 인스턴스 생성
1. ‘인스턴스 실행 템플릿’ 페이지에서 템플릿 행의 오른쪽에 있는 **인스턴스 생성**을 선택합니다.
<dx-alert infotype="explain" title="">
인스턴스 실행 템플릿의 **기본 버전** 설정을 사용하여 기본적으로 인스턴스를 생성합니다. 템플릿 행 앞의 <img src="https://main.qcloudimg.com/raw/bc4f9debb43518cdf0e1552f76f9b7fa.png" style="margin:-3px 0px">을(를) 클릭하고 펼쳐진 리스트에서 다른 버전을 선택하여 인스턴스를 생성할 수도 있습니다.
</dx-alert>
2. ‘CVM’ 생성 페이지의 ‘설정 정보 확인’ 단계에서 **기존 설정 비교**를 선택하고 ‘기존 설정 비교’ 팝업 창에서 인스턴스와 인스턴스 실행 템플릿의 차이를 확인할 수 있습니다.
3. 오류가 없는지 확인한 후 ‘<Tencent Cloud 서비스 약관> 및 <환불 규정>을 읽고 이에 동의함’을 선택하고, **활성화**를 클릭합니다.





### 인스턴스 실행 템플릿 삭제
1. ‘인스턴스 실행 템플릿’ 페이지에서 삭제할 인스턴스 실행 템플릿 행의 오른쪽에 있는 **삭제**를 선택합니다.
2. **삭제** 팝업 창에서 **확인**을 클릭합니다.
