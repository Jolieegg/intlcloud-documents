## 1\. **CONSIDERAÇÕES**

Este Módulo se aplica se você usar o recurso Bastion Host (“**Recurso**”). Este Módulo está incorporado ao Contrato de Processamento de Dados e Segurança localizado no (“[DPSA](https://intl.cloud.tencent.com/document/product/301/17347 )”). Os termos utilizados, mas não definidos nesse Módulo, devem ter o significado dado a eles no DPSA. No caso de qualquer conflito entre o DPSA e esse Módulo, esse Módulo será aplicado conforme a inconsistência.
