## 步骤4：安装部署 SSL 证书

>!
> 
> 使用一键 HTTPS 功能，您无需进行繁琐的 SSL 证书部署操作，即可帮助您实现从 HTTP 到 HTTPS 的能力升级。

1. 重新登录 [SSL 证书控制台](https://console.cloud.tencent.com/ssl)，进入 “我的证书” 管理页面。

2. 选择您需要安装的证书，单击操作栏的**下载**。

3. 下载证书后，将证书解压后安装至您的云服务资源中。
