桑基图是一种特定类型的流图，用于描述一组值到另一组值的流向。防火墙目的与源IP流量统计等场景适用。

## 图表配置

### 通用配置



| 配置项   | 说明                         |
| -------- | ---------------------------- |
| 基础信息 | 图表名称：设置图表的显示名称，可为空。                                 |
| 桑基图     | 方向：设置桑基图的显示方向。       |
